# 📊 Complete Exploratory Data Analysis (EDA) & Data Cleaning

A comprehensive Data Analysis project containing complete **Exploratory Data Analysis (EDA)** and **Data Cleaning** of three real-world datasets using **Python**, **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**.

This project was completed as part of a university Data Analysis assignment. All three datasets were analyzed in a **single Jupyter Notebook**, following a structured workflow from data preprocessing to visualization and insight generation.

---

## 📁 Datasets Analyzed

### 1️⃣ Loan Eligibility Dataset

This dataset analyzes factors that influence loan approval decisions.

**Major Analysis**

* Initial data inspection
* Missing value treatment
* Duplicate checking
* Data type verification
* Univariate analysis of every feature
* Loan approval analysis
* Credit History analysis
* Income analysis
* Correlation analysis
* Outlier detection
* Data visualizations
* Key business insights
